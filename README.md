# Wise Street theme
![alt text](https://raw.githubusercontent.com/lewiseman/assets/master/wise-street.png "Wise Street")


This theme is a combination of [SynthWave '84](https://marketplace.visualstudio.com/items?itemName=RobbOwen.synthwave-vscode "SynthWave '84") and [Tokyo Night](https://marketplace.visualstudio.com/items?itemName=enkia.tokyo-night "Tokyo Night")


I love SynthWave '84 text colors
## Installation

### Visual Studio Marketplace
1. Open **Extensions** sidebar in VS Code.
2. Search for `Wise Street Theme`.
3. Click **Install**.

### Open VSX Registry
1. Make sure you are using a VS Code compatible editor (like VSCodium) or have configured VS Code to use Open VSX.
2. Search for `Wise Street Theme`.
3. Click **Install**.
